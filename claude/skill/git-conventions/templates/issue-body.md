# Issue Body Template

```markdown
## Description

- Clear, concise explanation

## Steps to Reproduce _(bugs only)_

1. Step one

## Expected Behavior

- What should happen

## Actual Behavior _(bugs only)_

- What actually happens

## Proposed Solution _(optional)_

## References _(optional)_

- Related to #<num>
```

Issue title format: `<type>(<scope>): <short description>` - max 50 chars.
